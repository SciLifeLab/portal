Field conditional states installation instruction
-----------------------------------------------------------------------------

	* Download the module and place in your /sites/all/modules/ or 
      wherever you normally install your contrib modules.
    * Create new entity fields (at least one checkbox/radio or select list as 
      control field)
    * On the field setting page go to Conditional States tab
    * Select state type (visible, invisible, enabled, disabled or required)
      from the list, select your control field and available control values
      for selected control field.
    * In case of multiple selection for control values, select condition type
      (AND or OR)
      
Requirements     
-----------------------------------------------------------------------------
	Drupal 7.14 or higher
